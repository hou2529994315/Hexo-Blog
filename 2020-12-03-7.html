c# 简介

语法

  • 类和接口

    • Class
    • Interface
    • C#不支持多继承，所以需要多继承的时候需要用到接口

  • 属性、方法、事件

  • 函数可以当作值来对待

    • 委托

  • 支持纯模式

    • 避免使用值可变的变量

  • 类型安全

    • 静态类型
    • 支持动态类型
    • 强类型

内存管理

  • CLR公共语言运行时
    • GC垃圾回收器
  • C#中仍然存在指针
    • 但是通常情况下不需要使用指针
    • 使用指针时需要用 unsafe标记

平台支持

  • 基于.NET Core可以运行于各个平台

  • .NET/.NET Core的核心时CLR:公共语言运行时

  • CLR与语言无关

支持C#的框架

  • .NET Framework
  • .NET Core
  • Unity
  • UWP
  • Sql Server
  • …

C#基础知识

  • .NET Core 包括

    • CLR 公共语言运行时
    • FCL 常用函数（文件操作，http连接等等）

  • 第三方库的平台
    Nuget

  • C#编译器可以把.cs结尾的文件编译成Assembly

  • Assembly是.NET Core里的包装和部署单元

  • Assembly可以是应用程序，也可以是库

    • —-.exe—-
    • .dll (在bin文件夹下)

obj里是一些临时文件，bin和obj文件都可以随意删除

  • 标识符

    • 强行使用关键字做标识符可以用@ 如 @class

  • 上下文关键字

  • 静态类

    • 非静态类里可以有静态属性和非静态属性
    • 静态类内只能有静态属性和非静态属性
    • 静态类不能创建实例

  • 隐式转换的条件

    • 编译器保证可以转换成功
    • 转换不会有损失

  • 显式转换的条件

    • 编译器不能保证可以转换成功
    • 转换会有损失

分类
值类型 总是复制赋值
引用类型 赋值时仅复制地址值
泛型
指针

  • 内置类型

    • 除decimal以外的内置类型都叫做原始类型
    • System.IntPtr和System.UIntPtr也是原始类型

  • 数值类型

    • 数值赋值时可以使用下划线,例如：
      int million = 1_000_000;
    • 类型推断（小数->double,整数->可容纳该值的最小类型）
    • checked关键字可令加减乘除抛出异常,来检查溢出
      ``` git
      checked
      {
      int d = a * b;
      }
    • 1.0/0.0 = Infinity
    • 1.0/-0.0 = -Infinity
    • 0.0/0.0 = NaN
      ```

  • NaN 不等于任何值，包括NaN本身

  • 使用object.Equals()时，两个NaN相等
    验证某个值是否为NaN,可以使用float.IsNan(),double.IsNan()
    decimal非常适合财务计算

Null

可空值类型除了可以表示正常数据，还可以表示null,使用方法
Nullable<int> indexOfM = null
简写 int？ indexOfM = null

空字符串 null、空 “”、空白字符串” (space) “

  • Nullable<T>的常用属性和方法
    • .HasValue //null:false
    • .Value//T的值
    • .GetValueOrDefault()//T的值或者T的默认值
    • .GetValueOrDefault(默认值)//T的值或者指定的默认值
