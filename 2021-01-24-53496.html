MySql(4) 常用函数

MySQL
发布日期:   2021-01-24
文章字数:   647

函数的调用：

SELECT 函数名(实参列表) FROM 表;

分类：
1、单行函数：CONCAT,LENGTH,IFNULL
2、分组函数：用于统计，又称为统计函数、聚合函数、组函数

单行函数

字符函数

  1. length,获取参数值的 字节个数（与编码方式有关）

    SELECT LENGTH('John');
SELECT LENGTH('测试中文132');

  2. concat,拼接字符串

    SELECT CONCAT(last_name,',',first_name) 姓名 FROM employees;

  3. upper,lower, 大小写转换

    SELECT UPPER('john');
SELECT LOWER('JOHN');

  4. substr，substring，截取字符串，两个函数一样

    SELECT SUBSTR('123456',4) out_put;  # 截取第四个到结尾的字符。
SELECT SUBSTR('123456',4, 1) out_put;  # 截取从第四个字符开始的指定长度的字符。

  5. 姓名的首字符大写，其它字符小写，并用下划线拼接
    ``` mysql
    SELECT
    CONCAT(

     UPPER(SUBSTR( last_name, 1, 1 )),
 LOWER(SUBSTR( last_name, 2 )),
 '_',
 UPPER(SUBSTR( first_name, 1, 1 )),
 LOWER(SUBSTR( first_name, 2))) out_put

    FROM employees;

  6. instr,返回字串字串第一次在主串中出现的位置索引，如果找不到，返回0

    SELECT INSTR('123456789','456') AS out_put;

  7. trim,去前后的空格（中间的不去），可以去除指定的字符

    SELECT TRIM('                  23 4       ') AS out_put;
SELECT TRIM('aa' FROM 'aavssacsaaaaa')

  8. lpad,用指定字符左填充指定长度

    SELECT LPAD('12354',10,'*');

  9. rpad，右填充

    SELECT RPAD('12354',10,'*');

  10. replace, 替换

    SELECT REPLACE('123456123456123789123','123','321');

    数学函数

  11. round 四舍五入

    SELECT ROUND(-1.55);# -2

  12. ceil,向上取整

    SELECT CEIL(-1.05); # -1

  13. floor,向下取整

    SELECT FLOOR(-1.05); # -2

  14. truncate,截断，小数点后保留几位

    SELECT TRUNCATE(3.1415926,3); # 3.141

  15. mod，取余

    SELECT MOD(11,3);
SELECT 10%3;

日期函数

  1. now,返回当前系统日期+时间

    SELECT NOW();

  2. curdate，返回当前系统的日期

    SELECT CURDATE();

  3. curtime，返回当前系统的时间

    SELECT CURTIME();

  4. 获取指定部分的日期。年、月、日、时、分、秒

    SELECT YEAR(NOW()) 年;
SELECT YEAR('1998-05-26')
SELECT YEAR(hiredate) 年 FROM employees;

  5. str_to_date,将日期类型的字符转换为指定格式的日期
    格式:

序号 格式符 功能
1 %Y 四位的年份
2 %y 2位的年份
3 %m 月份( 01,02..11,12 )
4 %c 月份( 1,2..11,12 )
5 %d 日( 01,02… )
6 %H 小时( 24小时制)
7 %h 小时( 12小时制)
8 %i 分钟( 00,01…59 )
9 %s 秒( 00,01,…,59 ) 
# 查询入职日期为1992-4-3的员工信息
SELECT * FROM employees WHERE hiredate = '1992-4-3';
SELECT * FROM employees WHERE hiredate = STR_TO_DATE('4-3-1992','%c-%d-%Y');
  1. date_format,将日期转换为字符
    SELECT DATE_FORMAT(NOW(),'%y年%m月%d日') AS out_put;

其它函数

SELECT VERSION();#查看数据库版本号
SELECT DATABASE();# 查看当前使用的数据库
SELECT USER(); # 查看当前登录的用户

流程控制函数

  1. if函数，三元运算符的效果

    SELECT IF(10<9,'小','大');

  2. case函数，switch-cased的效果
    格式：
    case 要判断的字段或表达式/不写
    when 常量1/条件 then 要显示的值1或语句1;
    when 常量2/条件 then 要显示的值2或语句2;
    else 要显示的值n或语句n;
    end

# 查询员工的工资，要求
部门号=30，显示的工资为1.1倍
部门号=40，显示的工资为1.2倍
部门号=50，显示的工资为1.3倍
其他部门，显示的工资为原工资
SELECT
    salary 原始工资, department_id,
CASE department_id 
    WHEN 30 THEN salary * 1.1 
    WHEN 40 THEN salary * 1.2 
    WHEN 50 THEN salary * 1.3 
    ELSE salary 
    END AS 新工资 
FROM    employees;

分组函数

  • 分类
    • sum 求和
    • avg 平均值
    • max 最大值
    • min 最小值
    • count 计算不为null的数据的个数

  1. 简单使用

    SELECT SUM(salary) FROM employees;
SELECT AVG(salary) FROM employees;
SELECT MIN(salary) FROM employees;
SELECT MAX(salary) FROM employees;
SELECT COUNT(salary) FROM employees;

  2. 特点

  • 支持的数据类型：

    • sum, avg：数值型
    • max,min，count：数值、字符、日期（任何类型）

  • 是否忽略null值

    • 均忽略null值

  1. 和distinct搭配去重

    SELECT SUM(DISTINCT salary) FROM employees;

  2. count函数详解

    SELECT COUNT(commission_pct) FROM employees;# 结果：35   统计某一字段
SELECT COUNT(*)    FROM employees;# 结果：107  统计总行数，有任一字段不为空都会被统计
SELECT COUNT(1) FROM employees; # 在原表基础上加了一个常量值，统计个数

  3. 效率：
    MYISAM存储引擎下， count()的效率最高
    INNODB存储引擎下， count(    )和count(1)的效率并列第一
    一般用count(*)

注意：可以和分组函数以同查询的字段有限制,要求是group by后的字段。
